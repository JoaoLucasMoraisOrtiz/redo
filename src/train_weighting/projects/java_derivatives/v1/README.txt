Partial derivative and calculus utilities implemented in Java.

Highlights:
- Partial derivative estimator with forward and central schemes.
- Gradient calculator built on top of the partial derivative component.
- Single-variable derivative calculator using finite differences.
- Numerical integration with trapezoidal and Simpson's rule implementations.
- Basic arithmetic utility supporting add, subtract, multiply, divide, power, and roots.
- Sample univariate and multivariate functions for experimentation.
- CLI entry point showcasing all operations.
